# 🚀 SAI - Atalhos para abrir o sistema

Três formas de abrir o **SAI - Sistema de Alerta Inteligente** direto com duplo-clique:

## 🪟 Windows
- **`SAI.bat`** — duplo-clique que abre o navegador com o sistema. Você pode:
  - Colocar esse arquivo na Área de Trabalho
  - Criar um atalho (Botão direito → Enviar para → Área de Trabalho)
  - Fixar na Barra de Tarefas
- **`SAI.url`** — atalho web puro do Windows. Duplo-clique abre o navegador padrão direto, sem janela preta.

## 🍎 Mac / 🐧 Linux
- **`SAI.sh`** — duplo-clique (ou via terminal) abre o navegador.
- Antes de usar pela primeira vez, dê permissão:
  ```bash
  chmod +x SAI.sh
  ```
  E depois duplo-clique ou `./SAI.sh`

## 🌐 URL do sistema
```
https://projeto-ao-vivo.preview.emergentagent.com
```

## 📱 Celular
Abra a URL no navegador do celular, clique no menu → **"Adicionar à tela inicial"**. Vai criar um ícone que parece um app, e abre direto ao tocar.

---

## 🎯 Recomendado para uso diário (Windows)
1. Copie o arquivo **`SAI.url`** para a sua Área de Trabalho
2. Clique com o botão direito nele → **Propriedades** → **Alterar ícone...**
3. Escolha um ícone (ex: um escudo ou sino de alerta)
4. Pronto! Agora você tem um ícone bonito que abre seu SAI direto.
